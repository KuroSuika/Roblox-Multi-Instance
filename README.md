Made by KuroSuika

Video on how it works - 

## INSTRUCTIONS:
1. Double-click RMI.exe
2. When Windows asks for permission, click "Yes"
3. The program will automatically detect and close the ROBLOX_singletonEvent handle
4. Click "Stop Process" or close window to exit and stop the automatic handle closing.

## REQUIREMENTS:
- Windows 10/11 (64-bit)
- Administrator privileges

## NOTE:
- Some antivirus software may flag this tool (false positive)
- The program only closes a specific Roblox handle, it does not modify files
- If you still have concerns, throw the .exe into any virus scanner like VirusTotal
